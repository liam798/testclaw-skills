# TestClaw Skills

Open-source Agent skills for TestClaw.

## Skills

- `testclaw-cli`: use the TestClaw CLI and platform workflow for device management, app operations, test assets, suite execution, evidence collection, and reports.

## Layout

```text
testclaw-cli.skill
testclaw-cli/
  SKILL.md
  agents/
  references/
  scripts/
```

The main TestClaw repository references this repository as a Git submodule at `skills/`.
